# hk-pop
## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| value | 绑定值 | boolean | - | false |
| showMask | 是否显示遮罩层 | boolean | - | true |
| maskClickDisabled | 遮罩层是否可点击关闭pop | boolean | false |
| showCancel | 是否显示取消按钮 | boolean | - | true |
