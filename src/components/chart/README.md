# hk-chart

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| type | 图表类型 | string | line/bar/radar/polarArea/pie/doughnut/bubble | line |
| data | 图标显示数据 |	object | - | - |
| options | 配置项 | object | - | - |

**注: 具体参数配置详见[chart.js](http://www.chartjs.org) type，data，options都是new Chart(ctx, config)方法的第二个参数config**
