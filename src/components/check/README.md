# hk-check

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| activeColor | 选中颜色 | string | - | #3AA4F7 |
| fontColor | 未选中颜色 | string | - | #666 |
| text | 文案 | string | - | - |  |
| icon | 选中图标 | string | - | fa-check-circle-o |
| offIcon | 未选中图标 | string | - | fa-circle-o |
| disabled | 是否禁用 | boolean | - | false |
| active | 是否选中 | boolean | - | false |

## Events
| 事件名称 | 说明 | 回调参数 |
|--- | --- | --- |
| click | 鼠标点选事件 | 改变后的值 |
