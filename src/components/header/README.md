# hk-header

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| title | 标题 | string | - | - |
| leftIcon | 左边按钮图标 | string | - | - |
| leftText | 左边按钮文字 | stirng | — | - |
| rightIcon | 右边按钮图标 | stirng | - | - |
| rightText | 右边按钮文字 | stirng | — | - |

## Slot
| name | 说明|
| --- | --- |
| - | 标题 |
| left | 左边按钮(包含图标和文字) |
| leftIcon | 左边图标 |
| leftText | 左边文字 |
| right | 右边按钮(包含图标和文字) |
| rightIcon | 右边图标 |
| rightText | 右边文字 | 

## Events
| 事件名称 | 说明 |	回调参数 |
|--- | --- | --- |
| click-left | 左边按钮点击事件 | - |
| click-right | 右边按钮点击事件 | - |
