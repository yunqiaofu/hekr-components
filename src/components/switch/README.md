# hk-switch

## Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- | --- | --- | --- | --- |
| onColor | 选中颜色 | string | - | #3AA4F7 |
| offColor | switch类型未选中颜色 | string | - | #C0BFBF |
| text | 文案 | string | - | - |  |
| disabled | 是否禁用 | boolean | - | false |
| active | 是否选中 | boolean | - | false |

## Events
| 事件名称 | 说明 | 回调参数 |
|--- | --- | --- |
| click | 鼠标点选事件 | 改变后的值 |
